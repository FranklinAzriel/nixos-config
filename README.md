## Installation
1. Clone this repository
```
git clone https://github.com/FranklinAzriel/nixos-config /etc/nixos
```
2. Rebuild the system
```
nixos-rebuild --upgrade --flake path:/etc/nixos boot
```
3. Reboot your system
```
systemctl reboot
```
